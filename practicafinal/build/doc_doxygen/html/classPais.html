Detailed Description

T.D.A Pais.

Una instancia del tipo de dato Abstracto Pais nos permite almacenar datos sobre un Pais en un único dato

Estos datos son punto geográfico, nombre y bandera.

El TDA también proporciona herramientas para su manipulación y uso

Para poder usar el TDA Pais hay que incluir el fichero

#include <Pais.h>

TIPO DE REPRESENTACIÓN: Conjunto formado por un Punto y dos String que representa posición geográfica, nombre y pathname de la bandera del pais respectivamente
