Detailed Description

T.D.A Almacen de Rutas.

Una instancia del tipo de dato abstracto Almacen de Rutas nos permite almacenar una colección de rutas y acceder a ellos con facilidad mediante el uso de identificadores de ruta guardados como clave

El TDA proporciona herramientas para su manipulación y uso

Para poder usar el TDA Paises hay que incluir el fichero

#include <Almacen_Rutas.h>

TIPO DE REPRESENTACIÓN:

Colección de claves con objetos asociados en los que la claves son string y los objetos asociados son objetos de la clase Ruta.

La relación entre clave y objeto es la siguiente:

String = Ruta.GetCode
