Detailed Description

T.D.A Ruta.

Una instancia del tipo de dato Abstracto Ruta nos permite almacenar datos sobre un conjunto de Puntos que forman una Ruta y ponerle un código identificatorio

Cada punto es un objeto de la clase Punto

El TDA también proporciona herramientas para su manipulación y uso

Para poder usar el TDA Ruta hay que incluir el fichero

#include <Ruta.h>

TIPO DE REPRESENTACIÓN: Conjunto formado por una colección de puntos geográficos y un string que representa el recorrido de la ruta en un mapa y su identificador respectivamente.
