Detailed Description

T.D.A Paises.

Una instancia del tipo de dato Abstracto Paises nos permite almacenar una colección de paises y acceder a ellos con facilidad

EL TDA proporciona herramientas para su manipulación y uso

Para poder usar el TDA Paises hay que incluir el fichero

#include <Paises.h>
